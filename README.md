# 🍱 급량비 자동 계산 프로그램 (Streamlit)

급량비 계산의 번거로움을 줄이고, 복잡한 수당 정산을 빠르고 정확하게 처리하기 위해 제작된 자동 계산 프로그램입니다. 

## 🔍 코드 투명성 및 검증 안내 (Transparency)

* **보안 및 검증 목적**: 본 저장소는 급량비 자동 계산 프로그램의 소스코드를 투명하게 공개하여, 코드 내에 악성코드나 유해한 스크립트가 포함되어 있지 않음을 증명하고 검증받기 위한 목적으로 운영됩니다.
* **안전성 확인**: 모든 소스코드는 공개되어 있으므로 누구나 직접 로직을 검토하고 안전성을 확인할 수 있습니다.

---

## ✨ 주요 기능
* **자동 계산**: 근무 시간 및 조건에 따른 급량비 즉시 산출
* **편의성**: 복잡한 수당 기준을 반영한 직관적인 대시보드 화면 제공
